The Bridge Project

Partners: Michael Brodsky, Noah Hightower, Nathan Kantarov

Principles of Engineering
Started: 1/28/2021 Ended: 2/14/2021

It was our assignment to design a bridge that would allow for a truck to pass through the bridge. As a group, we wanted to make the cheapest bridge that could still support the truck without breaking. We all spent our time making the best bridge that fits within these limits. I was able to build the final bridge that was one of the cheapest by using a building technique of muscles. Muscles acted as a stronger beam that can support more weight because it divided the stress into two pathways instead of one, allowing the high stress areas to take on more stress than they could.

MIT App Inventor Moblie App: PaintPotPro

Partner: Renxiang(James) Wu

CSP
Started: 9/23/19 Ended: 10/9/19

It was our objective to make an application for the andriod tablet we had in class, and we had to make the application in MIT App Inventor. MIT App Inventor is a block-based coding website where the user can create different apps using different combinations of blocks. The twist was that we had to make a program out of one the existing tutorials and "remix" or upgrade the tutorial program already existing on their website. We decided to choose the painting app tutorial and brainstormed many ideas to implement before going in to code. This project taught us how to split work evenly and how to be as efficient and orgainized as possible so that others could easily understand our code.
